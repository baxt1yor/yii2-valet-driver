# yii2-valet-driver
Yii2 Valet Driver

## yii2-app-basic

```
valet link app-name
http://app-name.test
```

## yii2-app-advanced

```
cd backend
valet link admin.simple
http://admin.simple.test

cd frontend
valet link simple
http://simple.test
```

and

config assets URL if you use the link for the assets folder and set the link

```
cd assets
valet link assets.simple
http://assets.simple.test
```
